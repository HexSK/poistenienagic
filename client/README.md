# Prehlad SQL skriptov

## Odporucane poradie spustenia
1. `tables.sql`
2. `insert.sql`
3. `functions.sql`
4. `procedures.sql`
5. `triggers.sql`
6. `views.sql`
7. `select.sql` (kontrolne a zadane dotazy)

## Ucel suborov
- `tables.sql`: zakladna struktura databazy
- `insert.sql`: vstupne data.
- `functions.sql`: 2 funkcie pozadovane zadanim.
- `procedures.sql`: 2 procedury pozadovane zadanim.
- `triggers.sql`: logika triggerov (validacia + evidencia historie).
- `views.sql`: 3 pohlady s pouzitim `INNER`, `LEFT` a `RIGHT` joinov.
- `select.sql`: pozadovane selecty (`WHERE + LIKE + IN`) a agregacne dotazy (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`).
